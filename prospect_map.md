# Prospect Map — France
**Cible : Freelances, indépendants, petites agences (1–10 personnes)**

---

## Pourquoi cette cible ?
- Problème universel (factures impayées, tréso tendue)
- Décision rapide : pas de comité, pas de procurement
- Budget disponible (€200–350/mois = une demi-journée de travail)
- Pas d'équipe finance interne → ils font tout eux-mêmes

---

## Sources de prospection (gratuites)

### 1. SIRENE / INSEE (gratuit)
- **URL :** https://www.data.gouv.fr/fr/datasets/base-sirene-des-entreprises-et-de-leurs-etablissements-siren-siret/
- Base officielle de TOUTES les entreprises françaises
- Filtrer par : code NAF (activité), taille (0–9 salariés), région, date de création
- **Codes NAF pertinents :**
  - 7022Z — Conseil pour les affaires et management
  - 7311Z — Activités des agences de publicité
  - 6201Z — Programmation informatique
  - 6920Z — Activités comptables (éviter — ils connaissent déjà le sujet)
  - 7410Z — Activités spécialisées de design
  - 6311Z — Traitement de données
- **Limite :** pas d'emails ni de contacts nominatifs

### 2. Pappers.fr (gratuit)
- **URL :** https://www.pappers.fr
- Interface simple sur la base SIRENE
- Recherche par activité, localisation, chiffre d'affaires
- Export limité en gratuit

### 3. Societeinfo.com (freemium)
- **URL :** https://www.societeinfo.com
- Données légales + segmentation avancée
- Couvre bien les TPE
- Plan payant : ~€50–150/mois pour exports

### 4. Pharow (payant, recommandé)
- **URL :** https://www.pharow.com
- Meilleure base B2B française en 2026
- Données SIRENE + LinkedIn + emails enrichis
- Prix : ~€150–300/mois selon volume
- **À utiliser à partir du mois 2** quand vous avez validé le modèle

## Sources de prospection (gratuites + emails)

### 5. Malt.fr
- **URL :** https://www.malt.fr
- Plateforme de freelances française
- Profils publics avec noms, activité, localisation
- Trouver l'email via Hunter.io ou LinkedIn

### 6. Google Maps
- Recherche : "agence web [ville]", "consultant marketing [ville]"
- Trouver le site → trouver l'email de contact
- Manuel mais gratuit et ciblé localement

### 7. Annuaire des Agences (gratuit)
- **URL :** https://www.sortlist.fr ou https://www.agencesdefrance.fr
- Listes d'agences avec sites et contacts

## Process de prospection semaine 1

1. Aller sur Pappers.fr → filtrer : agences / consultants / freelances tech/marketing, créés 2020–2024, Île-de-France ou grande ville
2. Extraire 50 noms d'entreprises
3. Aller sur leur site → noter l'email de contact
4. Si pas d'email sur le site → utiliser Hunter.io avec le domaine
5. Importer dans Brevo
6. Envoyer la séquence 3 emails (voir pitch_emails.md)
7. Objectif : 50 emails/semaine → 2–4 réponses → 1 appel
